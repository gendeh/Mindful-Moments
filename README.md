# Mindful Moments - AI-Powered Journaling Companion

An intelligent journaling companion that uses AI to make self-reflection effortless and insightful, addressing the common barriers to maintaining consistent journaling habits.

## Problem Statement

While journaling offers well-documented mental health benefits, many people struggle with:
- Blank page anxiety - Not knowing what to write
- Lack of guidance - Especially for beginners  
- No meaningful insights - Journals become mere event logs rather than growth tools

## Solution Overview

Mindful Moments combines the therapeutic benefits of journaling with AI-powered guidance and analysis to create a truly intelligent companion for mental wellness.

### Key Features

- Dynamic AI Prompts: Fresh, personalized reflection questions generated by Gemini AI
- Real-time Sentiment Analysis: Automatic emotional tone detection using advanced NLP
- Emotional Trends Dashboard: Visual charts showing mood patterns over time
- Privacy-First Design: Secure AI processing with minimal data retention

## Design Inspiration

The design was inspired by the need to make journaling more accessible and less intimidating. Traditional journaling apps focus on text input but don't address the psychological barriers that prevent consistency. By integrating AI as a gentle guide rather than a replacement for human reflection, Mindful Moments creates a supportive companion that encourages daily practice while providing meaningful insights.

The visual design emphasizes calm and approachability, using soft colors and clean typography to reduce anxiety. The emotional trends chart transforms abstract feelings into concrete data, helping users see patterns they might otherwise miss.
### Core Components

- **Flask Web Framework**: Provides the web server and routing infrastructure
- **Google Gemini AI**: Powers prompt generation and sentiment analysis
- **Bootstrap CSS Framework**: Ensures responsive, professional UI design
- **Matplotlib**: Generates emotional trend visualizations
- **Jinja2 Templates**: Handles dynamic content rendering

## Implementation Details

### Backend (Flask + Python)

The application uses Flask for its simplicity and extensibility. Key implementation choices:

- **Modular Functions**: Separate concerns with dedicated functions for AI interaction, data processing, and visualization
- **Error Handling**: Graceful degradation when AI services are unavailable
- **In-Memory Storage**: Simple data persistence for prototype, easily replaceable with database
- **Environment Configuration**: Secure API key management using environment variables

### AI Integration (Gemini)

Gemini 2.5 Flash Lite was selected for its balance of performance and capability:

- **Prompt Generation**: Uses natural language prompts to create contextually appropriate reflection questions
- **Sentiment Analysis**: Leverages Gemini's understanding of emotional language for accurate tone detection
- **Real-time Processing**: Provides immediate feedback without noticeable latency

### Frontend (Bootstrap + HTML)

The user interface prioritizes simplicity and emotional safety:

- **Minimalist Design**: Reduces cognitive load and anxiety
- **Responsive Layout**: Works across device sizes
- **Accessible Colors**: Warm, calming color palette
- **Progressive Enhancement**: Core functionality works without JavaScript

### Data Flow

1. User loads page → Flask generates AI prompt → Display in UI
2. User writes entry → Submit to Flask → Send to Gemini for analysis
3. Store results → Update visualization → Display updated dashboard

## Installation

1. Clone repository: `git clone https://github.com/gendeh/Mindful-Moments.git`
2. Create virtual environment: `python -m venv venv`
3. Activate environment: `source venv/bin/activate`
4. Install dependencies: `pip install flask google-generativeai matplotlib python-dotenv`
5. Configure API key: Create `.env` file with `GEMINI_API_KEY=your_key`
6. Run application: `python -m flask run`
7. Access at: http://127.0.0.1:5000

## Usage

1. Open application to view AI-generated daily prompt
2. Write thoughts in the text area
3. Submit entry to analyze sentiment and update trends
4. View emotional patterns in the dashboard
5. Return daily to build journaling consistency
